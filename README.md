# Taido-Trainer

TAIDO TRAINER コンセプト
目的
一人でも、目的に応じた質の高い反応練習を可能にする。
相手の動きを想定して、外部刺激に瞬時に反応する能力を、号令役のいない自主練でも再現。
「回数指定モード」と「無限モード」により、質を高める集中訓練から限界に挑むスタミナ訓練まで、個人の目的に合わせた最適な環境を提供。
このアプリで鍛えられること
反応速度: 合図（音）に瞬時に反応し、初動を最速にする。
集中力（回数指定）: 回数がわかるからこそ、1回ごとの質を高める。
精神力（無限）: 終わりのない状態の中、技の速さ、正しい構え、質の高い技を維持する。

実装機能ロードマップ
【Ver.1】ベーシック（済）: ランダム間隔の号令、エラー処理
【Ver.2】モードセレクト（済）: 回数指定、無限モード、集中を高めるスマート画面
【Ver.3】トレーニング支援: 3・2・1のカウントダウン、無限モード用タイマー
【Ver.4】躰道要素: 「旋・運・変・捻・転」のランダム表示、五つの技の音声読み上げ
最終的なビジョン
「躰道の自主練を、もっと実戦的に、もっとスマートに。」
場所や時間を選ばず、常に実戦を意識できる“デジタル指導者”へ。「1人でできること」の幅を増やす。
