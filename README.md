# Taido-Trainer
瞬発力を鍛えよう😊

🥋 TAIDO TRAINER コンセプト

目的

一人でも、質の高い反応練習を可能にする。

躰道では、自分のタイミングで動くのではなく、相手の動きや号令などの外部刺激に対して瞬時に反応する能力が重要である。しかし、自主練では号令役がいないことが多く、そのような反応練習を行うことは難しい。

本アプリは、不規則なタイミングで合図を出すことで、自主練でも実戦に近い反応練習を実現することを目的とする。

⸻

このアプリで鍛えられること

* 反応速度
    合図を聞いて瞬時に技へ移る能力。
* 集中力
    いつ合図が来るか分からない状況で集中を維持する能力。
* 構えの維持
    待ち時間が一定でないため、気を抜かずに構え続ける習慣が身につく。
* 自主練の質の向上
    一人でも、号令役がいるような練習を行える。

⸻

コンセプト

「一人でも、練習相手がいるように。」

⸻

今後実装したい機能

Ver.1

* ✅ ランダム間隔の号令
* ✅ 入力エラー処理
* 🔄 3・2・1・開始カウントダウン
* 🔄 カウントダウンにホイッスル音追加

⸻

Ver.2

* 練習時間を可視化するタイマーの表示

⸻

Ver.3

* 旋・運・変・捻・転のランダム表示
* 五つの種類の音声読み上げ

⸻

最終的なビジョン

「躰道の自主練習を、もっと実戦的に。」

部員一人ひとりが場所や時間を選ばず、実戦を意識した反応練習を行える”デジタル練習相手”を目指す。
